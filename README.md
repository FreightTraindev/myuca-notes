# myuca-notes

## Markdown Cheat Sheet for everyday use:

### Headings:

(#) H1
(##) H2
(###) H3
(####) H4

Examples:
# H1
## H2
## H3
#### H4

### Text Formatting:

(star) Italic (star) or (underscore)italic(underscore)

(2S)bold(2S)

(3S) Bold italic (3S)

~~ Strikethrough ~~ 

Examples:

*Italic*

*Bold*

***Bold Italic***

~~Strikethrough~~
